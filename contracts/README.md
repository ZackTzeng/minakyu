# Minakyu zkApp

A Minakyu smart contract that verifies the authenticity of a signed API response (generated by the code in the [oracle](../oracle/))

This template uses TypeScript.

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
